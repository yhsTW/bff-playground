# 사전 준비

1. 노드 버전 확인

```shell
node --version
```

2. yarn 설치

```shell
npm install --global yarn
```

3. yarn 버전 확인

```shell
yarn --version
```

# 설치

1. 레포지토리 클론

```shell
git clone https://github.com/yhsTW/bff-playground.git
```

2. 패키지 설치

```shell
yarn install
```

3. 환경 변수 설정

    [.env.example](https://github.com/yhsTW/bff-playground/blob/main/.env.example)을 읽고 환경 변수를 설정합니다.
